# MarketPulse

MarketPulse is a containerised ASP.NET Core application demonstrating modern software engineering practices in a financial technology context.

## Technology Stack

- ASP.NET Core Web API
- Entity Framework Core
- PostgreSQL
- Docker & Docker Compose
- Swagger / OpenAPI
- React (planned)
- GitHub Actions (planned)

## Goals

- Demonstrate clean architecture and layered design
- Expose RESTful APIs for financial market data
- Integrate PostgreSQL using Entity Framework Core
- Provide production-style health monitoring
- Containerise all services
- Implement CI/CD with GitHub Actions
